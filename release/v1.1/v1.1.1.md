# v1.1.1 — 稳定性修复

> **发布日期**: 2026-06-18  
> **基于**: v1.1.0  
> **下载**: `AI热点日报.exe` (47MB)

---

## 🐛 Bug 修复

| 问题 | 说明 |
|------|------|
| **点击报告链接误杀后台** | 所有链接改为新标签页打开（`target="_blank"`），不再触发 `beforeunload` 关闭进程 |
| **同日期多文档全部高亮** | 用文件名替代日期做唯一识别键，点击只高亮选中的那一个 |
| **自定义文档无法删除** | 新增 `/api/reports/file/<filename>` CRUD 端点 |
| **DELETE 误删当天全部报告** | 移除危险的模糊匹配兜底，仅精确文件名删除 |
| **爬虫线程永久转圈** | 新增 `finally` 保护 + 200 秒超时兜底 |
| **exe 无法加载配置文件** | 启动时强制从用户目录重载 `config.local.yaml` |
