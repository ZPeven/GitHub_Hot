# v1.1.0 — Deep Space Observatory 🛰️

> **发布日期**: 2026-06-18  
> **下载**: `AI热点日报.exe` (47MB)  
> **最低配置**: Windows 10/11 64-bit · 256MB+ 内存 · 无需 Python/mamba

---

## 📦 使用方式

1. 下载 `AI热点日报.exe`，放到任意文件夹
2. 双击运行 → 浏览器自动打开 Deep Space Observatory 界面
3. 首次自动弹出设置面板，填写 Token 和代理端口（也可跳过）
4. 点 `▶ TRANSMIT` 开始抓取 AI 热点
5. 所有报告保存在 exe 同目录 `reports/` 下
6. 关闭浏览器页面即结束程序

## 🐛 Bug 修复

| 问题 | 说明 |
|------|------|
| **关闭页面后后台进程不退出** | 关闭浏览器标签页时自动结束后台进程 |
| **设置面板无法保存配置** | `use_proxy` 布尔值导致保存报错；脱敏 token 被错误覆盖 |
| **exe 无法正确识别配置文件** | 启动时重新从用户目录加载 `config.local.yaml` |
| **爬虫线程可能永久转圈** | 新增 `finally` 保护 + 200 秒超时兜底 |
| **报告标题多余 `{#llm}` 锚点** | 改用标准 `<a id>` 标签，兼容 `marked.js` |
| **同日期多文档全部高亮** | 用文件名替代日期做唯一识别键 |
| **自定义文档无法删除** | 新增文件名级别 CRUD 端点 |

## ✨ 新功能

| 功能 | 说明 |
|------|------|
| 🎨 **Deep Space Observatory UI** | 宇宙深空主题，粒子星场 + 青蓝/琥珀双霓虹配色 |
| ⚙ **应用内设置面板** | 点击工具栏齿轮填写 API Token / 代理 / 翻译配置 |
| 🔢 **端口号简写** | 输入 `10101` → 系统自动补全 `http://127.0.0.1:10101` |
| 📝 **新建文档** | 侧边栏 `+ NEW`，弹窗输入标题创建空白笔记 |
| 🗑 **删除文档** | 侧边栏 `DEL`，确认后删除当前文档 |
| 👋 **首次使用引导** | 无配置文件时自动弹出设置面板 |
| 🌐 **中日英双语翻译** | 非中文标题和摘要自动翻译为中文 |
| 🔗 **标题附带原文链接** | 每条标题可点击跳转原文/原论文/原项目 |
| ✏ **Markdown 实时预览** | 左侧编辑源码，右侧实时渲染，Ctrl+S 保存 |
| 📊 **历史报告管理** | 左侧列表点击切换，抓取完成后自动刷新 |
| 💾 **设置持久化** | 配置自动写入 `config.local.yaml`，下次启动自动加载 |

## 🗂 文件结构

```
你的文件夹/
├── AI热点日报.exe          ← 双击运行
├── config.local.yaml        ← 配置文件（自动生成）
├── reports/                 ← 报告目录（自动创建）
│   ├── 2026-06-18_AI_Hotspot_Report.md
│   └── 2026-06-18_Custom.md
└── crawler.db               ← 历史数据库（自动创建）
```

## ⚙ 配置说明

| 配置项 | 必填 | 说明 |
|--------|------|------|
| GitHub Token | 否 | 不填使用匿名限额（60次/小时），填写后 5000次/小时 |
| DeepSeek API Key | 否 | 不填则跳过双语翻译，报告仅显示原标题 |
| 代理端口 | 否 | 中国大陆用户填写，留空则直连 |
